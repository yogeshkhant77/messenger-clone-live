# messenger-clone-live
full mern stack live messenger
A Node.js + Express backend for a real-time chat application using MongoDB, Mongoose, and Pusher for instant message updates.
This project powers a Messenger clone, handling message storage and live updates between users.

## Features
RESTful API for sending and retrieving messages

Real-time updates with Pusher

Database connection using MongoDB + Mongoose

CORS-enabled Express server

Auto-reload using Nodemon for development

## Installation
# Clone the repository
git clone https://github.com/yogeshkhant77/messenger-clone-live.git
cd messenger-backend

# Install dependencies
npm install
## Run the Server
npm start
